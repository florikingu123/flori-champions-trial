## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Backend**: PHP with MySQLi
- **Database**: MySQL
- **AI**: Google Gemini API
- **Server**: XAMPP (Apache + MySQL + PHP)

## Local Access
